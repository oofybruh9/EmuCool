# Commands used by a lot of popular emulators

| Emulator    |  OS   | command                                                                 |
|:---         | :---: |                                                                     ---:|
| RetroArch   | W/L   | retroarch -f -L %core% %rom$                                            |
| RetroArch   | macOS | /Applications/RetroArch.app/Contents/MacOS/RetroArch -f -L %core% %rom% |
| Snes9x      | All   | snes9x %rom%                                                            |
| FCEUX       | All   | fceux --fullscreen %rom%                                                |
| ZSNES       | W/L   | zsnes -s %rom%                                                          |
| Mesen       | All   | mesen %rom% --fullscreen                                                |
| Dolphin     | All   | dolphin-emu -c -b -e %rom%                                              |
| MelonDS     | All   | melonds -f %rom%                                                        |
| PCSX2       | All   | pcsx2 -fullscreen %rom%                                                 |
| RPCS3       | All   | rpsc3 --fullscreen %rom%                                                |
| Duckstation | All   | duckstation -fullscreen -batch -- %rom%                                 |
| Cemu        | All   | cemu -f -g %rom%                                                        |
| PPSSPP      | All   | ppsspp --fullscreen --escape-exit %rom%                                 |
||||
